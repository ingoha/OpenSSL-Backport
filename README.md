OpenSSL-Backport
================

OpenSSL Backport 1.0.1k for Squeeze<br/>
<br/>
Only Build:<br/>
$ dpkg-buildpackage<br/>
<br/>
Just Install:<br/>
$ dpkg -i *.deb<br/>
<br/>
Test and list it!<br/>
$ openssl version<br/>
OpenSSL 1.0.1k 8 Jan 2015<br/>
